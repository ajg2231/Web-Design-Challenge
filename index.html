HOME PAGE

This is a weather visualizations website. To analyze weather data, the OpenWeatherMap API was used to collect data on a multitude of different cities.

These cities were then plotted using MatPlotLib. The main facets of weather data looked at were: temperature, cloudiness, wind speed, and humidity. This website provides links to the source data, visualizations, and explanations of any statistical patterns that were witnessed.
